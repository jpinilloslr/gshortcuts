gshortcuts

Note: This project is in early development. Please check back in a few days for updates.

Import and export your GNOME custom keyboard shortcuts with ease.

Get Started GitHub

Features

Import & Export

Backup and restore your custom GNOME shortcuts effortlessly.

Merge Strategies

Override or merge shortcuts with existing ones.

Multi-Format Support

Import and export shortcuts in YAML and JSON formats.

Lightweight

No dependencies, simple CLI interface.

Installation

curl -sSL https://gshortcuts.jpinillos.dev/install.sh | bash

Usage

# Export shortcuts to a YAML file
  gshortcuts export shortcuts.yaml

# Export shortcuts to a JSON file
  gshortcuts export shortcuts.json

# Import shortcuts with merge strategy
  gshortcuts import shortcuts.yaml --strategy=merge

# Override existing shortcuts
  gshortcuts import shortcuts.yaml --strategy=override
